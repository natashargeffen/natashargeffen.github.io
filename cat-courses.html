CatCourses Website

This site was created with the goal of making life easier for University of Vermont students. Using PHP and SQL, I created a database that allows students to search and review UVM courses and professors, so that students can use this tool when picking courses for upcoming semesters.

Students can fill out a form about a course they have taken...

...and search for reviews that have been written about a course they are interested in.
